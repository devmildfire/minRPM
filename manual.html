Manual

This manual will help you integrate AADS code on your site to publish ads within your ad-units.

First, generate ad-unit HTML code for one ot more ad-units.

You do this by providing your ad-unit id, size and fallback code (if needed)

If you need to add more ad-units, just click "Add Ad-Unit" button

After you are done with the inputs, click "Generate" button to get the integration code for every one of your ad-units
